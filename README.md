# 💬 ClassChat – TCP Client-Server Chat System

## 📌 Overview
ClassChat is a TCP-based client-server chat application developed using Python.  
It allows multiple clients to communicate with each other through a centralized server.

## 🚀 Features
- TCP socket communication
- Multi-client concurrent handling using threading
- JSON-based structured message transmission
- Client management system
- Error handling for offline users
- Real-time messaging

## 🛠 Technologies Used
- Python
- Socket Programming
- Threading
- JSON
- TCP/IP Protocol

## 🧠 How It Works
1. Server starts and listens for connections.
2. Clients connect to the server.
3. Messages are sent in JSON format.
4. Server forwards messages to intended receiver.
5. Multiple clients can communicate simultaneously.
